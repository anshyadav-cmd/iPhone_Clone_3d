> (React three fiber)

# Project [@Live](https://iphone-clone-3d.pages.dev) 

building a 3D/webflow design of the iPhone.

## Installation

- Clone the Repo
  ```bash
  git clone git@github.com:anshyadav-cmd/iPhone_Clone_3d.git
  ```
- Install all dependencies
  ```bash
  npm install
  ```
- Start the development server
  ```bash
  npm run dev
  ```
- Open your browser and navigate to `http://localhost:5173`

==Note Use a compatible Browser that can run WebGL2 to open the project. Tested on **Safari** and **Chrome** it works fine but not on **Brave Browser**. ‚úåüèª
